Stego tools (for pictures)

Cheatsheet - Stego tools for pictures Link to heading

Stegoveritas Link to heading

Stegoveritas can be found here : https://github.com/bannsec/stegoVeritas

Create several altered images (contrast/colors/brightness)

Run with common options :

$ stegoveritas -exif -meta -xmp -carve -imageTransform -BruteLSB -extractLSB -trailing massacre.png

Steghide Link to heading

Available in most of distro’s repos.

Common run :

steghide extract -sf file.jpg

Stegcracker Link to heading

Stegcracker can be found here : https://github.com/Paradoxis/StegCracker

Performs a steghide bruteforce :

stegcracker file.jpg /usr/share/wordlist/rockyou.txt
